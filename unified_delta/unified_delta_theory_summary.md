```

```

## 5. トポロジー追跡（Q_Λ）

### 原子振動の位相
```
u = A × exp(iφ)

A: 振幅 → δ（Lindemann比）
φ: 位相 → Q_Λ（巻き数、トポロジー）
```

### プラケット巻き数
```
Q = (1/2π) ∮ ∇φ · dl

格子上の最小ループ（プラケット）で位相変化を積分
Q = 0: 正常
Q ≠ 0: 渦（転位の芯）
ΔQ ≠ 0: リコネクション！
```

### 破壊の検出
```
材料破壊 = Q のジャンプの連鎖
         = 渦のリコネクション
         = トポロジー変化
```
